# Kidney DTSim

A discrete-time simulation of kidney organ allocation in Germanny. The model couples patient and donor agent populations, applies Eurotransplant allocation logic, and produces transplant outcomes and post-transplant risk estimates.

## Installation

Installing it from the repository root

```bash
pip install -e "."
```

or

```bash
pip install git+https://github.com/na55imK/kidney_dtsim.git
```
